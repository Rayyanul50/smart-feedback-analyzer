# 🤖 Smart Feedback Analyzer

An intelligent feedback analysis tool powered by IBM Watson Natural Language Understanding. Analyze customer feedback, reviews, and comments with sentiment analysis and keyword extraction.

![Python Version](https://img.shields.io/badge/python-3.7+-blue.svg)
![IBM Watson](https://img.shields.io/badge/IBM%20Watson-NLU-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![GUI](https://img.shields.io/badge/GUI-Tkinter-orange)

## 🌟 Features

- **Real-time Sentiment Analysis**: Analyze feedback and get instant sentiment (Positive/Negative/Neutral)
- **Keyword Extraction**: Automatically identifies key topics and phrases
- **Batch Processing**: Upload CSV files to analyze multiple feedback entries
- **Export Results**: Save analysis results to CSV for further processing
- **User-friendly GUI**: Simple and intuitive graphical interface
- **Free to Use**: Works with IBM Watson's free tier

## 📋 Prerequisites

- Python 3.7 or higher
- IBM Cloud account (free tier available)
- Internet connection for API calls
