## 🌟 Features

- **Real-time Sentiment Analysis**: Analyze feedback and get instant sentiment (Positive/Negative/Neutral)
- **Keyword Extraction**: Automatically identifies key topics and phrases from feedback
- **Batch Processing**: Upload CSV files to analyze multiple feedback entries at once
- **Export Results**: Save analysis results to CSV for further processing and reporting
- **User-friendly GUI**: Simple and intuitive graphical interface built with Tkinter
- **Free to Use**: Works with IBM Watson's free tier (1000 API calls per month)

## 📋 Prerequisites

- Python 3.7 or higher
- IBM Cloud account (free tier available)
- Internet connection for API calls
